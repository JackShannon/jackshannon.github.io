    Summit

    I created this playable demo to demonstrate the techniques of my 3D game engine called ‘Indigo’. It uses modern shader based OpenGL programming and is written in C++. I am constantly researching and adding features to this engine so it is to be seen as a work in progress.

    Techniques featured

    • Deferred Lighting with a Per-Pixel Blinn-Phong Lighting Model
    • Normal Mapping
    • Screen-Space Ambient Occlusion
    • Cascading Shadow Maps
    • Terrain (Brute-forced chunks with Frustum Culling)
    • Physics (using the Bullet Physics Engine)
    • Skybox
    • Fog
    • Billboard grass, using Quadtrees for procedural placement and culling
    • Alpha blending and alpha testing
    • High Level rendering architecture
    • Render Queue, sorting and culling of redundant state changes
    • Robust shader system similar to DirectX effects
    • Scene management
    • Proprietary file formats/loaders
